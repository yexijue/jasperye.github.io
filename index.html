第四章 社会认知

一.社会知觉

  1. 社会知觉的特点:
    1. 互动性:社会认知是双向的
    2. 选择性:人对于自己相关度较高的信息反应更强烈,对于强度大的刺激更易引起关注
    3. 防御性:个体对信息的知觉反应随着个人情绪状态不同而不同
    4. 完形性:消除认知失调
  2. 图式:认知者组织信息的内在心理机制(schemas)
    1. 种类:
      1. 自我图式
      2. 角色图式
      3. 个人图式
      4. 社会事件图式
    2. 作用:
      1. 帮助记忆
      2. 导致自动化推论
      3. 增加信息
      4. 包含情感
    3. 启动效应:
      1. 概念:最近使用的图式在接下来的不相关情境中再次被使用的现象
      2. 前提:先前激活的图式在新信息中是有意义的
      3. 内隐性社会认知:过去的经历影响个体对当前情境中社会客体的情感思想和行为,且个体对此无意识
  3. 认知偏差
    1. 光环效应：当认知者对一个人的某种特征形成积极印象之后，还倾向于据此推出该人其他方面的积极特征
    2. 正性偏差：认知者在谈论被认知者时，表达正面评价往往多于负面评价
    3. 内隐人格理论：运用少数已知特征判断他人的特点
    4. 相似性假设（虚假一致性偏差）：倾向认为认知者与自己相似，包括思考和行动。（以己度人）
  4. 首因效应：最初形成的印象不易改变，而且影响后来相关信息的解释（近因效应类似，最新形成的印象）
  5. 刻板印象
    1. 定义：人们对信息类别化处理，对某个群体或某一类人形成的一种概括而固定的看法
    2. 范畴化：通过了解某事物和其他事物的异同点来理解这一事物
  6. 社会范畴理论 social categorization
    1. 范畴异质性和同质性——外群同质性效应
    2. 范畴化无意识驱动：想想老年人就会让自己走得更慢
    3. 归因：人对行为原因的推论过程。观察者对行为过程所进行的因果解释和推论（一致性+一贯性+区别性）
    4. 对应推断理论（琼斯&戴维斯）：外显行为是由行为者的内在品质直接引起的（他纹身，不是好人）
  7. 协变理论：
    1. 一致性：该行为与他人的行为是否一致
    2. 一贯性：该行为与行为者在其他场合的行为表现是否一致
    3. 区别性：行为者对不同客体是否相同
  8. 韦纳的成败归因理论：

    内因外因、稳定不稳定是人们在进行归因时所考虑的两个维度，这两个维度互相独立，人们如何归因会影响今后的成就行为。

  9. 归因偏差
    1. 定义：认知者歪曲了原本正确的信息
    2. 基本归因错误：过高估计行为者内在因素的作用
    3. 行为者和观察者的归因差异：（宽于律己，严以待人）

      对他人的行为做倾向归因而将自己的行为推断为情境归因的倾向。简单来说，“别人做了某事是他自身的过失，而我做了同样的事情，是环境所迫而非自身所愿意的”。

    4. 自我防卫性归因：接受成功，拒绝失败；为了保护自己的自尊（甚至会自我设阻）

第六章 社会互动与人际关系

  1. 社会互动的主要类型
    1. 对称性互动：交换、合作、竞争、冲突
    2. 非对称性互动：暗示、模仿、感染
  2. 竞争的动机：人们试图在社会比较中确定自我价值的需求使人们倾向于超越别人，采用竞争的方式与他人发生关系（金字塔的社会结构）
  3. 囚徒困境：
  4. 目标手段相互依赖理论：

    个人的行为目标或手段 与 他人的行为目标或手段 之间如果存在相互依赖关系，其间就会发生相互作用，形成合作、竞争、独立三种关系。

  5. 影响群体成员的竞争与合作的因素：
    1. 沟通：沟通机会越多，合作可能越大
    2. 奖励结构：
      1. 竞争性共存：股票市场
      2. 合作性共存：鳄鱼与千鸟
    3. 价值观：
      1. 合作者：winwin
      2. 竞争者：零和博弈
      3. 个人主义者：理性经济人
  6. 人际关系的深度（相对应的还有广度）
    1. 社会渗透理论：个体之间从表面化的沟通到亲密的沟通而经历的关系发展过程。（表露水平意味着安全感和信任的水平）
  7. 人际关系的三维理论
    1. 舒茨（主动型与被动型）
      1. 包容需要：个人需要与别人接触、交往并建立与维持和谐关系的需要
      2. 支配需要：个体控制他人和被他人控制的需要
      3. 感情需要：在感情上与他人建立并维持良好关系的需要
    2. 6种人际关系取向：（包容、支配、感情）x（主动、被动）
    3. 童年期的人际需要满足程度以及由此形成的行为方式对成年后的人际关系有决定性影响
    4. 群体整合原则：包容、控制、感情
    5. 群体分解原则：难以包容、失去控制、感情不和
    6. 莱维特：（主从型、合作型、竞争型）C(3,1)+C(3,2)+C(3,3)+C(3,0)
  8. 沟通网络
    1. 正式沟通网络
    2. 非正式沟通网络
  9. 身体语言沟通
    1. 定义 ：非词语性的身体符号——包括目光、面部表情、身体运动与触摸、姿势与外貌等等
    2. 情绪变化首先反映在瞳孔变化
    3. 目光接触是最重要的体语沟通方式
    4. 面部表情可完成精细信息沟通
    5. 人在触摸和身体接触时情感体验最为深刻
  10. 距离
    1. 人际距离：交往过程中个体因为心理因素的影响与他人在地理空间上保持的距离
    2. 公众距离：公共场合的人际距离，3.7-7.6m，7.6m+

第五章 社会态度

  1. 态度
    1. 定义：人的社会态度是由个体的认知、情感和行为意向等心理成分所组成，与人的观念意识相联系并受其影响和指引的内在心理组织和系统。

      社会态度=Z（认知+情感+行为意向）+X（人的观念意识）

    2. 本质：人们对于客观社会存在的有意识地反映于映像，是人们有意识的认知活动和心理状态的综合表现和综合反应。
    3. 成分：
      1. 认知：我认为早睡对身体好
      2. 情感：我喜欢看电影
      3. 行为倾向：我赞成现在出门散步
    4. 维度
      1. 关联度：各要素之间相互关系和影响
      2. 一致性：各要素之间相互一致协调
      3. 中心度：各要素在态度中所具有的作用
    5. 态度——行为关系的决定因素
  2. 服从——认同——内化（凯尔曼）
    1. 服从：被动趋于对象，受外因控制
    2. 认同：主动趋于对象，主要受情感因素影响
    3. 内化：内在化，从内心相信并接受
  3. 认知相符理论之海德的平衡理论
    1. 重视人与人之间的相互影响对态度改变的作用
    2. 态度系统总是趋于平衡
    3. 费力最小原则：个体尽可能少地改变情感因素而维持态度平衡
    4. P——O——X 模式:P是自己，O是客体，X是影响因素（可以是人或事物）

      三种肯定，或者两种否定，一种肯定，则存在平衡状态。

  4. 认知相符理论之认知失调理论
    1. 认知失调是一个人的态度和行为等认知成分的相互矛盾，从一个认知推断出另一个对立的认知时而产生的不舒服感、不愉快的情绪。（我讨厌詹姆斯，我的好朋友是詹密）由于人们在心理上产生不适，不协调的存在将推动人们努力减少不协调。
      1. 解决方法：
        1. 改变态度
        2. 改变行为
        3. 增加认知
        4. 减少选择
    2. 过度理由效应：外在理由取代内在理由成为主要的行为支持力量，行为由内部控制转向外部控制（一个人每天唱歌，后来邻居给他钱；一段时间后，邻居不给他钱了，他也就停止了唱歌）
    3. 最简暴露效应：暴露频次与最初态度之间存在正向线性相关关系。（因为我们经常看到镜像自我，所以比起他拍照，我们更喜欢自拍照）
  5. 说服者本身的特性：
    1. 睡眠者效应：润物无声——当下被拒绝，实则已接受。信源的可信度对信宿态度的影响程度与时间的长度呈反比。
    2. 得失理论：人们倾向于对自己喜爱程度提高的人，不喜欢对自己喜爱程度降低的人。（++，–，-+，+-。四组里人们最喜欢第三组，最讨厌第四组，对一二组感受则一般）
      1. 外来和尚好念经
      2. 恋人平淡久了容易出现厌倦
      3. 人们喜欢新鲜感
  6. 偏见、刻板印象、歧视
    1. 偏见：刻板印象的一部分。否定、负面、更多的主观偏好与情感成分。
    2. 歧视：偏见的激化版——对偏见对象的恶意丑化、污名化
    3. 降低偏见：支持平等的社会规范，接触需要建立在平等的基础上，持续密切合作。

第三章 社会动机

一：社会动机以及相关研究

  1. 概念辨析：
  • 需要：个体赖以 生存和发展 所必须的条件
  • 内驱力：在需要的基础上 产生的 一种内部的 唤醒状态或紧张状态
  • 动机：引起和维持 有机体活动，并使活动朝向某一目标的心理过程叫做动机
  • 社会动机：动机+社会属性
  • 自我决定的三种基本心理需要：胜任+自主+归属
  • 人类的基本动机相同,但是满足动机的方式不同

二：社会动机的理论解释

  1. 本能论
  2. 非本能论:归因理论(韦纳成败归因理论)+社会学习理论(班杜拉自我效能感)

三.侵犯与挫折

  1. 挫折：

    个体在从事有目的的活动过程中，遇到障碍或干扰，致使个人目标无法实现、个人需要不能满足时，产生的一种消极的情绪状态。

  2. 侵犯类型
    • 敌意性:单纯报复
    • 工具性:为了目标
  3. 挫折-侵犯理论(多拉德)

    挫折→侵犯
    侵犯永远是挫折的一种后果

  4. 米勒的修正

    挫折→(侵犯+其他反应)

  5. 武器效应(伯克维茨)

    (挫折+侵犯线索)→侵犯 (手枪)

  6. 为了避免侵犯:
    • 宣泄假设:侵犯性精神能量是常数,实际或想象的侵犯行为都可以使其释放
    • 习得性抑制:替代攻击项目(打沙包)+社会规范抑制(道德+法律)
    • 避免 去个性化(个体自身的同一性意识下降,自我评价和控制自我水平 下降 的现象)(匿名,酗酒,嗑药)

社会信念和判断

社会世界的感知

启动（priming）

  • 记忆是一个相互联结的系统，而启动就相当于激活某些联结的特殊指令，类似于多出的线头。称之为启动效应。
  • 具身认知（embodied cognition）；身体感觉也会启动社会判断。例如：身体的温暖会强化社会温暖，社会排斥会让人感觉更寒冷

感知和解释事件

  • 我们的信念塑造了我们对于其他任何事物的解释

信念固着（belief perseverance）

  • 先灌输信念A，然后推翻A，大部分人仍然坚持A；先入为主
  • 人们执着于先前自己所持有的信念
  • 解释相反的观点，可以降低甚至消除信念固着

构建自己和世界的记忆

  • 误导信息效应（misinformation effect）：人们把误导性的信息整合在记忆中
  • 记忆系统并非储存室，而是会被现在影响，人们在回忆的时候会对其进行重构

反事实思维（counterfactual thinking）

  • 构建与事实相反的幻觉，给我们带来好的或者坏的感受

认识自我

自尊（self-esteem）

  • 定义： 我们对自我的全面评价
  • 自我图式（self-schemas）+ 可能自我（possible self）？
  • 自尊动机影响认知过程：
    * 高自尊：面对失败，高自尊的人会认为他人也和自己一样失败，并且夸大自己相对于他人的优越性，以维持自己的自我价值
    * 低自尊：也称安全型自尊，这类人自我价值感更为脆弱，会经历更多的压力、愤怒、人际关系问题

行动中的自我

自我的能量

  • 自我控制：自我控制会消耗有限的意志力
  • 自我概念会影响自我的行为

自我效能（self-efficacy）

  • 定义：我们感到有能力完成某项任务
  • 较高的自我效能可以让我们的表现更强
  • 另一个影响因素:你有没有控制住结果？
  • 与自尊的差异：相信自己有能力做一些事，这是自我效能；由衷地喜欢自己，这是自尊
  • 自我效能的反馈比自尊的反馈更佳

控制点（locus of control）

  • 定义：如何看待命运——由自己掌控（内部控制点），由外界掌控（外界控制点）
  • 相信内部控制点会带给我们更好的行为表现；但是这种好处是有局限性的

习得性无助（learned helplessness）与 自我决定（self-decision）

  • 习得性无助：人们尝试去改变i中状态但是毫无收获时产生
  • 自我决定：成功的控制+改善自我状态 共同支撑
  • 适度的自我决定可以提高老人的认知表现（养老院例子）
  • 过度的自由和自我决定反而会降低人们的幸福感（巧克力酱and星巴克饮料），即较小的选择范围更能带来满足感

自我服务偏差（self-serving bias）

自我服务归因（self-serving attributions）

  • 好的归因于自我，坏的归因于其他
  • 关于自我服务偏差的经典笑话：一个丈夫对妻子说：“如果咱们俩中的一个先去世，我想我会搬到巴黎去住。”

乐观与悲观

  • 乐观可以增加个体自我效能感，促进幸福和健康
  • 防御性悲观主义(defensive pessimism)，中国人常说的居安思危

虚假普遍性（false consensus effect）和 虚假独特性（false uniqueness effect）

  • 虚假普遍性：高估自己观点和弱点的普遍性；我们看到的并非世界本身，而是自己的影子；归因为我们采用不可信的样本（样本小且包括自我在内）
  • 虚假独特性：低估自己能力和品德的普遍性；在一些方面认为自己是小众的，为了完善自我形象

自我表露的管理(self-pre)

自我妨碍（self-handicapping）

  • 因为害怕失败，所以消极地表现，方便事后将失败归功于此；以此保护自己的自尊和公共形象

印象管理

  • 自我表露：人们想向外在的观众和内在的观众展现一种受赞许的形象。通常是无意识的
  • 自我监控（self-monitoring）：长时间有意识地自我表露，不断根据周围环境来调整自己的行为，以试图合群。变色龙
  • 高自我监控——行骗专家；低自我监控——榆木脑袋

既要相信自我效能的力量也要警惕自我服务偏差

正如哲学家帕斯卡尔说过：

任何一个单独的真理都是不充分的，因为世界是很复杂的。任何一个真理如果脱离了和它互补的真理，就只能算是部分真理而已。

环境变迁
