清朝

黄宗羲（1610-1695）

  1. 理想社会
    1. 形式：
      1. 君主一心为民
      2. 官员一心为民
      3. 法令规范为民
      4. 学校养士且议政
      5. 井田制的恢复
    2. 社会特点
      1. 托古的形式
      2. 民主主义色彩
      3. 乌托邦式
      4. 具有否定和批判现实的启蒙色彩（超前性）
  2. 论人性与欲望
    1. 一元论，认为人性是在与生俱来的人心中
    2. 性善论者，仁义礼智与生俱来，基本照搬孟子
    3. 人欲则是好逸恶劳，自私自利
  3. 论社会问题
    1. 社会习俗问题：仪式繁复，铺张浪费；尊佛敬巫，浪费三力
      1. 治本：一切从简，减少浪费；兴教化，对抗佛巫
      2. 治末：禁止工商业发展，则无所浪费（黄评价：这是错误的）
    2. 官吏危害民众：（解决办法，官位都由科考士人担任）
      1. 曲解法律谋私利
      2. 官僚队伍里有无赖
      3. 吏部对人事变动疏于管理
      4. 机要岗位出现世袭制
    3. 人才教育选拔与任用问题（解决办法：取士宽，用士严）
    4. 土地集中赋税过重
      1. 积累莫返之害：两税法，税负太重（每次赋税改革都加重人民的负担）
      2. 所税非所出之害：统一收银的赋税制度让百姓苦不堪言
      3. 田土无等第之害：土地产出不一，但是税负却一样

黄宗羲定律：历史上的赋税改革不止一次，但每次赋税改革后。农民负担下降段时间后又涨到比改革前更高的水平。

  1. 评价
    1. 对现实问题认识深刻，但是解决方法软弱
    2. 因阶级不成熟，所以思想不成熟
    3. 有很强的幻想性——缺少方法论
    4. 极具进步性，但存在阶级局限性
    5. 具有民主启蒙内容，堪称先驱

    2. 廉耻具有社会规范功能
    3. 孝悌具有社会规范功能

  3. 论社会问题及解决方法（极具现实主义）

      1. 赋税征银（因地决定征收银子还是谷物）
      2. 贫富悬殊（减少剥削，折衷主义）
      3. 社会风俗（廉耻、孝悌）
        1. 人情：人表里不一
        2. 佛道：浪费三力
        3. 官僚：官僚主义
      4. 生员制度（举人以下成为准官僚）（清理生员）
        1. 生员过多
          1. 弱化政府权威
          2. 加重农民负担
          3. 结党
          4. 破坏取士渠道
        2. 所学无用
  4. 理想社会模式
    1. 注重人民日用饮食（非常独特）
    2. 讲人道、明人伦、厚风俗（寄希望于道德）
    3. 安宁的农业社会（开历史倒车）
    4. 君主分权（近代民主主义）
    5. 宗法制（托古）
    6. 民众有舆论监督权
  5. 评价：

    顾炎武的社会思想是进步的，其理想社会构想也带有浓重的时代气息，反映出随着社会发展，中国的社会思想正在不断冲破旧的格局而不断觉醒和向着近代民主主义进步的趋向。

两宋时期

  1. 时代特点
    1. 解决儒家内部问题：否定和超越汉唐经学形态
    2. 对抗外来思想
  2. 理学（理和气，两大支柱）
    1. 理学——应运而生（实质：以儒学为主体，涵泳三教思想的基础上创建的以伦理为本体的哲理体系）
      1. 理学：两宋诸子所创，以理作为宇宙最高本体
      2. 新儒学：外表不变，内核已被加以改造
      3. 道学：明道作为宗旨
    2. 过程
      1. 初创：周敦颐、邵雍
        1. 周敦颐：
          1. 太极图说（无极而太极）。吸收佛教的宇宙图式和哲学思辨，为理学家出入佛道开辟了道路
          2. 秩序情节：万一各定，大小有定
          3. 理欲三字
            1. 定：定之以中正仁义
            2. 主：主静
            3. 诚：立诚；即真实客观
          4. 孔颜乐处的个体修养功夫
      2. 奠基：张载、二程
        1. 张载：（1020-1078）
          1. 儒学——佛道——批判佛老——回归儒学
          2. 气论：
            1. 整个世界统一于气
              1. 太虚与万物相互转换
              2. 一切自然现象就是气的聚散运动；散而为太虚
              3. 太虚和气不是先后生成关系，而是本体与现象的关系
            2. 思想价值：
              1. 驳斥贵无论和唯心论
              2. 探讨天人关系
          3. 人性论：
            1. 气构成万物，所以人也是气构成的。气的本性就是人的本性
            2. 人性：天地之性+气质之性
              1. 天地之性是与生俱来的
              2. 气质之性是后天形成的（攻取之性）
              3. 两者相对，前者善；后者可善可不善，乃恶之来源，需要改正
                1. 如何改正
                  1. 善反
                  2. 学
          4. 民吾同胞的社会救助论
            1. 一切人都是同胞兄弟，一切物都是同伴
            2. 并非人人平等
            3. 天地万物和人类的等级差别是天然合理的，人类要无条件承认。特别是道德上遵守
            4. 人生态度：“存，吾顺事；没，吾宁也。”
        2. 二程：
          1. “理”
            1. 承认气化论，反对气本论
            2. 对理的界定：
              1. 物理和物之理
              2. 事理、人理
              3. 统一的理
            3. 理欲关系：
              1. 性即理
              2. 人性二重性（道心和人心）：所以有 天人交战
                1. 天理和人欲对立
                2. 存天理，灭人欲
                3. 存理灭欲，灭的是不符合封建礼教的私欲
                4. 对妇女的桎梏：饿死事小，失节事大
      3. 成熟：朱熹
        1. 朱熹：（1130-1200）
          1. 理本气末
            1. 理是最高范畴，各个事物都有理
              1. 理在先，气在后
              2. 理为本：理生气，理气相依；理与气，理支配气
            2. 万理一源，理一分殊
              1. 宇宙间只是一个太极，万物各有一个太极
            3. 人性论
              1. 天命之性和气质之性
                1. 天命之性：仁义礼智在人心的体现（理）（论天地之性，则专指理言）
                2. 气质之性：理在人身上体现，同人所受的气相结合构成（理与气杂而言之）
              2. 道心、人心
                1. 道心：感物而动产生的合乎天理的心和情
                2. 人心：感物而动受到物欲牵累的不善的心和情
              3. 存天理，灭人欲
      4. 瓦解：王阳明
  3. 评价理学
    1. 礼治秩序的重建。
      1. 只承认人伦关系网的存在，否定独立个体价值，塑造主奴根性
      2. 穷理灭欲，用礼制秩序压迫妇女
    2. 内圣经世的路线高扬。造就政治全然从属于道德的统治纲领
      1. 重义轻利
      2. 空疏学风：不注重实际知识，只注重内在精神追求
    3. 塑造中华民族重道德责任的文化性格
      1. 孔颜乐处
      2. 民吾同胞
      3. 浩然正气
    4. 理学和中国社会发展
      1. 宋明文化：新的意识形态。文化从外向转向内向
      2. 两宋：专制权力的扩张和强化。治世先治君
      3. 外王转向内圣：“平时袖手谈心性，临危一死报君王”

    3. 仁
      1. 贵仁
      2. 将外在的礼变为内在的仁

孟子

      2. 心善乃性善的根据（恻隐之心）
      3. 四心，道德的肇端（仁义礼智）

    2. 天命观
      1. 自然之天
      2. 主宰之天：通过民意体现;福祸取决于自身；具有客观必然性涵义
    3. 养气论

第四章 社会认知

一.社会知觉

  1. 社会知觉的特点:
    1. 互动性:社会认知是双向的
    2. 选择性:人对于自己相关度较高的信息反应更强烈,对于强度大的刺激更易引起关注
    3. 防御性:个体对信息的知觉反应随着个人情绪状态不同而不同
    4. 完形性:消除认知失调
  2. 图式:认知者组织信息的内在心理机制(schemas)
    1. 种类:
      1. 自我图式
      2. 角色图式
      3. 个人图式
      4. 社会事件图式
    2. 作用:
      1. 帮助记忆
      2. 导致自动化推论
      3. 增加信息
      4. 包含情感
    3. 启动效应:
      1. 概念:最近使用的图式在接下来的不相关情境中再次被使用的现象
      2. 前提:先前激活的图式在新信息中是有意义的
      3. 内隐性社会认知:过去的经历影响个体对当前情境中社会客体的情感思想和行为,且个体对此无意识
  3. 认知偏差
    1. 光环效应：当认知者对一个人的某种特征形成积极印象之后，还倾向于据此推出该人其他方面的积极特征
    2. 正性偏差：认知者在谈论被认知者时，表达正面评价往往多于负面评价
    3. 内隐人格理论：运用少数已知特征判断他人的特点
    4. 相似性假设（虚假一致性偏差）：倾向认为认知者与自己相似，包括思考和行动。（以己度人）
  4. 首因效应：最初形成的印象不易改变，而且影响后来相关信息的解释（近因效应类似，最新形成的印象）
  5. 刻板印象
    1. 定义：人们对信息类别化处理，对某个群体或某一类人形成的一种概括而固定的看法
    2. 范畴化：通过了解某事物和其他事物的异同点来理解这一事物
  6. 社会范畴理论 social categorization
    1. 范畴异质性和同质性——外群同质性效应
    2. 范畴化无意识驱动：想想老年人就会让自己走得更慢
    3. 归因：人对行为原因的推论过程。观察者对行为过程所进行的因果解释和推论（一致性+一贯性+区别性）
    4. 对应推断理论（琼斯&戴维斯）：外显行为是由行为者的内在品质直接引起的（他纹身，不是好人）
  7. 协变理论：
    1. 一致性：该行为与他人的行为是否一致
    2. 一贯性：该行为与行为者在其他场合的行为表现是否一致
    3. 区别性：行为者对不同客体是否相同
  8. 韦纳的成败归因理论：

    内因外因、稳定不稳定是人们在进行归因时所考虑的两个维度，这两个维度互相独立，人们如何归因会影响今后的成就行为。

  9. 归因偏差
    1. 定义：认知者歪曲了原本正确的信息
    2. 基本归因错误：过高估计行为者内在因素的作用
    3. 行为者和观察者的归因差异：（宽于律己，严以待人）

      对他人的行为做倾向归因而将自己的行为推断为情境归因的倾向。简单来说，“别人做了某事是他自身的过失，而我做了同样的事情，是环境所迫而非自身所愿意的”。

    4. 自我防卫性归因：接受成功，拒绝失败；为了保护自己的自尊（甚至会自我设阻）

第五章 社会态度

  1. 态度
    1. 定义：人的社会态度是由个体的认知、情感和行为意向等心理成分所组成，与人的观念意识相联系并受其影响和指引的内在心理组织和系统。

      社会态度=Z（认知+情感+行为意向）+X（人的观念意识）

    2. 本质：人们对于客观社会存在的有意识地反映于映像，是人们有意识的认知活动和心理状态的综合表现和综合反应。
    3. 成分：
      1. 认知：我认为早睡对身体好
      2. 情感：我喜欢看电影
      3. 行为倾向：我赞成现在出门散步
    4. 维度
      1. 关联度：各要素之间相互关系和影响
      2. 一致性：各要素之间相互一致协调
      3. 中心度：各要素在态度中所具有的作用
    5. 态度——行为关系的决定因素
  2. 服从——认同——内化（凯尔曼）
    1. 服从：被动趋于对象，受外因控制
    2. 认同：主动趋于对象，主要受情感因素影响
    3. 内化：内在化，从内心相信并接受
  3. 认知相符理论之海德的平衡理论
    1. 重视人与人之间的相互影响对态度改变的作用
    2. 态度系统总是趋于平衡
    3. 费力最小原则：个体尽可能少地改变情感因素而维持态度平衡
    4. P——O——X 模式:P是自己，O是客体，X是影响因素（可以是人或事物）

      三种肯定，或者两种否定，一种肯定，则存在平衡状态。

  4. 认知相符理论之认知失调理论
    1. 认知失调是一个人的态度和行为等认知成分的相互矛盾，从一个认知推断出另一个对立的认知时而产生的不舒服感、不愉快的情绪。（我讨厌詹姆斯，我的好朋友是詹密）由于人们在心理上产生不适，不协调的存在将推动人们努力减少不协调。
      1. 解决方法：
        1. 改变态度
        2. 改变行为
        3. 增加认知
        4. 减少选择
    2. 过度理由效应：外在理由取代内在理由成为主要的行为支持力量，行为由内部控制转向外部控制（一个人每天唱歌，后来邻居给他钱；一段时间后，邻居不给他钱了，他也就停止了唱歌）
    3. 最简暴露效应：暴露频次与最初态度之间存在正向线性相关关系。（因为我们经常看到镜像自我，所以比起他拍照，我们更喜欢自拍照）
  5. 说服者本身的特性：
    1. 睡眠者效应：润物无声——当下被拒绝，实则已接受。信源的可信度对信宿态度的影响程度与时间的长度呈反比。
    2. 得失理论：人们倾向于对自己喜爱程度提高的人，不喜欢对自己喜爱程度降低的人。（++，–，-+，+-。四组里人们最喜欢第三组，最讨厌第四组，对一二组感受则一般）
      1. 外来和尚好念经
      2. 恋人平淡久了容易出现厌倦
      3. 人们喜欢新鲜感
  6. 偏见、刻板印象、歧视
    1. 偏见：刻板印象的一部分。否定、负面、更多的主观偏好与情感成分。
    2. 歧视：偏见的激化版——对偏见对象的恶意丑化、污名化
    3. 降低偏见：支持平等的社会规范，接触需要建立在平等的基础上，持续密切合作。

  1. 概念辨析：
  • 需要：个体赖以 生存和发展 所必须的条件

  • 社会动机：动机+社会属性
  • 自我决定的三种基本心理需要：胜任+自主+归属
  • 人类的基本动机相同,但是满足动机的方式不同

  1. 本能论
  2. 非本能论:归因理论(韦纳成败归因理论)+社会学习理论(班杜拉自我效能感)

三.侵犯与挫折

  1. 挫折：

    个体在从事有目的的活动过程中，遇到障碍或干扰，致使个人目标无法实现、个人需要不能满足时，产生的一种消极的情绪状态。

  2. 侵犯类型
    • 敌意性:单纯报复
    • 工具性:为了目标
  3. 挫折-侵犯理论(多拉德)

    挫折→侵犯
    侵犯永远是挫折的一种后果

  4. 米勒的修正

    挫折→(侵犯+其他反应)

  5. 武器效应(伯克维茨)

    (挫折+侵犯线索)→侵犯 (手枪)

  6. 为了避免侵犯:
    • 宣泄假设:侵犯性精神能量是常数,实际或想象的侵犯行为都可以使其释放

    • 避免 去个性化(个体自身的同一性意识下降,自我评价和控制自我水平 下降 的现象)(匿名,酗酒,嗑药)

社会信念和判断

社会世界的感知

启动（priming）

  • 记忆是一个相互联结的系统，而启动就相当于激活某些联结的特殊指令，类似于多出的线头。称之为启动效应。
  • 具身认知（embodied cognition）；身体感觉也会启动社会判断。例如：身体的温暖会强化社会温暖，社会排斥会让人感觉更寒冷

感知和解释事件

  • 我们的信念塑造了我们对于其他任何事物的解释

信念固着（belief perseverance）

  • 先灌输信念A，然后推翻A，大部分人仍然坚持A；先入为主
  • 人们执着于先前自己所持有的信念
  • 解释相反的观点，可以降低甚至消除信念固着

构建自己和世界的记忆

  • 误导信息效应（misinformation effect）：人们把误导性的信息整合在记忆中
  • 记忆系统并非储存室，而是会被现在影响，人们在回忆的时候会对其进行重构

启发式判断（heuristics）：心理捷径

  • 代表性启发式判断（representativeness heuristic）

反事实思维（counterfactual thinking）

  • 构建与事实相反的幻觉，给我们带来好的或者坏的感受
  • 人们经常地未作为而非不作为感到歉意
