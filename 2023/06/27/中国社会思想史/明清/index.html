明代

王守仁

  1. 理想社会模式（人人有良知）
    1. 以天地万物为一体
    2. 人人各尽其心，促进社会幸福（各司其职，发挥才智）
    3. 人人同心同德，亲如一家（人人都以天下为己任）
  2. 模式特点
    1. 责任：以天下为己任
    2. 职业：按照才德分配，各司其职
    3. 社会：不分你我，人人平等
  3. 评价：过于美好，空想乌托邦
  4. 致良知学说
    1. 良知是一种能力，即知是非
    2. 良知是天理和道，可画等号
    3. 良知是社会规范的内化，是一套社会价值系统
    4. 良知人人都有。
    5. 良知在于格物。
  5. 心学
  6. 社会组织方法：
    1. “十家牌法”：严密社区街道制度
    2. “乡约”，乡村村规

王艮（泰州学派）

李贽

  1. 一个异端对正统的批判（反对假大空）
    1. 孔子：拉下神坛，认为其奸诈虚伪
    2. 儒学：儒生都是伪君子，表面尊道学，背地求富贵
    3. 道学：空谈天理的虚伪学者
  2. 理想社会
    1. 兵、食为基础
    2. 万物并育而不相害。全社会成员共同发展
    3. 治理顺乎人性，少加约束
    4. 尊重个性，优胜劣汰
  3. 论社会交往思想
    1. 夫妇平等：正夫妇
    2. 男女平等：并无差异
    3. 婚姻平等：赞成改嫁
    4. 人人平等：没有凡圣
  4. 论社会交往思想
    1. 社会关系是一种交换关系
    2. 天下无真正的朋友关系，只有利益关系
    3. 真正的朋友以义为基础，不会轻易离弃
  5. 论欲望
    1. 衣食住行是基本欲望
    2. 追求个人欲望是共同的人性
    3. 欲望源于人的本性，不可阻挡
    4. 欲望是行为的动力
    5. 欲望出自天人，追求欲望也是自然之理
    6. 私欲出于禀赋，合乎自然
      1. 若无节制，反受其害
      2. 追求之旅，名利不可得兼
      3. 知足就是富有（思想的继承性）

清朝

黄宗羲（1610-1695）

  1. 理想社会
    1. 形式：
      1. 君主一心为民
      2. 官员一心为民
      3. 法令规范为民
      4. 学校养士且议政
      5. 井田制的恢复
    2. 社会特点
      1. 托古的形式
      2. 民主主义色彩
      3. 乌托邦式
      4. 具有否定和批判现实的启蒙色彩（超前性）
  2. 论人性与欲望
    1. 一元论，认为人性是在与生俱来的人心中
    2. 性善论者，仁义礼智与生俱来，基本照搬孟子
    3. 人欲则是好逸恶劳，自私自利
  3. 论社会问题
    1. 社会习俗问题：仪式繁复，铺张浪费；尊佛敬巫，浪费三力
      1. 治本：一切从简，减少浪费；兴教化，对抗佛巫
      2. 治末：禁止工商业发展，则无所浪费（黄评价：这是错误的）
    2. 官吏危害民众：（解决办法，官位都由科考士人担任）
      1. 曲解法律谋私利
      2. 官僚队伍里有无赖
      3. 吏部对人事变动疏于管理
      4. 机要岗位出现世袭制
    3. 人才教育选拔与任用问题（解决办法：取士宽，用士严）
    4. 土地集中赋税过重
      1. 积累莫返之害：两税法，税负太重（每次赋税改革都加重人民的负担）
      2. 所税非所出之害：统一收银的赋税制度让百姓苦不堪言
      3. 田土无等第之害：土地产出不一，但是税负却一样

黄宗羲定律：历史上的赋税改革不止一次，但每次赋税改革后。农民负担下降段时间后又涨到比改革前更高的水平。

  1. 评价
    1. 对现实问题认识深刻，但是解决方法软弱
    2. 因阶级不成熟，所以思想不成熟
    3. 有很强的幻想性——缺少方法论
    4. 极具进步性，但存在阶级局限性
    5. 具有民主启蒙内容，堪称先驱

顾炎武（1613-1682）

  1. 欲望论
    1. 人是自私自利的：由此出发，其认为财产私有制是解决纷争和促进生产的关键。（金子论）
    2. 批判官吏贪腐：提出以名遏制这一现象（人都爱名声）
      1. 劝学以克利欲
      2. 奖廉以克利欲
  2. 论社会规范
    1. 礼是根本的社会规范
    2. 廉耻具有社会规范功能
    3. 孝悌具有社会规范功能
    4. 法制具有危害性——国将亡，必多制；光靠法律无法大治
  3. 论社会问题及解决方法（极具现实主义）
    1. 社会问题
      1. 赋税征银（因地决定征收银子还是谷物）
      2. 贫富悬殊（减少剥削，折衷主义）
      3. 社会风俗（廉耻、孝悌）
        1. 人情：人表里不一
        2. 佛道：浪费三力
        3. 官僚：官僚主义
      4. 生员制度（举人以下成为准官僚）（清理生员）
        1. 生员过多
          1. 弱化政府权威
          2. 加重农民负担
          3. 结党
          4. 破坏取士渠道
        2. 所学无用
  4. 理想社会模式
    1. 注重人民日用饮食（非常独特）
    2. 讲人道、明人伦、厚风俗（寄希望于道德）
    3. 安宁的农业社会（开历史倒车）
    4. 君主分权（近代民主主义）
    5. 宗法制（托古）
    6. 民众有舆论监督权
  5. 评价：

    顾炎武的社会思想是进步的，其理想社会构想也带有浓重的时代气息，反映出随着社会发展，中国的社会思想正在不断冲破旧的格局而不断觉醒和向着近代民主主义进步的趋向。

王夫之
