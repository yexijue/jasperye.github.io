孔子

  1. 贡献：
    1. 整理古代文献
    2. 创立传统教育制度
    3. 创立儒家学派，“制度化儒家”：

      思想观念的制度化,即是在权力的支持下，将一种思想体系转化成一种法律、习俗、实践系统。最高形态是儒家的意识形态化。

  2. 思想体系
    1. 天命观
      1. 天命含义：
        1. 超越之天：宗教意义上的最终归宿（形而上学）
        2. 道德之天：道德意义的秩序和法则（举头三尺有神明）
        3. 自然之天：自然变化的过程和规律（四季节律）
        4. 偶然命运之天：（怨天尤人）
      2. 态度：对鬼神持存而不论、敬而远之的态度，其天命观带有人文色彩和理性精神
      3. 影响：形成了“取验务实，注重实践”的民族性格；遏制纯思辨色彩的哲学发展
    2. 礼
      1. 重礼
      2. 约束人们行为的准则
    3. 仁
      1. 贵仁
      2. 将外在的礼变为内在的仁
  3. 思想评价：

孟子

  1. 孟子的思想来源：

    孔子——曾子——子思——子思门人——孟子

  2. 思想体系：
    1. 性善说

      第一次明确揭示了关于人性的新的观念:人具有不同于动物或他物的特殊性,这就是道德性。

      1. 人性善
      2. 心善乃性善的根据（恻隐之心）
      3. 四心，道德的肇端（仁义礼智）
      4. 良知良能说
      5. 虽人性善，但仍有君子小人之分（因为物质不同环境不同）
    2. 天命观
      1. 自然之天
      2. 主宰之天：通过民意体现;福祸取决于自身；具有客观必然性涵义
    3. 养气论
      1. 存心—放心—尽心—养心—“存夜气”
    4. 理想王道的福利观：
      1. 基本标准：《孟子 梁惠王上》
      2. 制民恒产：统治者分地给无地之人，让他们能养活自己和家人
      3. 经济状况和道德状况联系起来（马斯洛需求层次理论）
      4. 井田制
      5. 救济穷弱：鳏寡孤独，国家和社会组织进行救助

荀子

  1. 思想来源：
    1. 战国后期社会变迁的剧烈性
    2. 总体性和交融性：诸子合流
    3. 经验主义：经验论
    4. 提出群学
  2. 群学
    1. 命题一：天人相分
      1. 自然之天：并无道德之天
      2. 天人分途：天道不干预人事
      3. 制天命而用之
      4. 天人分途的影响：
        1. 社会分析的出发点是人的身体，身体社会学的发端
        2. 彻底的人本主义者，天人互不相干
    2. 命题二：性伪之分
      1. 人性恶：性恶论
      2. 化性起伪：将先天的自然资质转化为后天人为的道德特性
      3. “涂之人皆可以为禹”人性可以改造，圣人与凡人在本性上是相同的
    3. 命题三：人能群
      1. 明分使群:社会起源与人的社会化（有脑子，所以可以结群合作）
    4. 命题四：隆礼重法

老子

  1. 道家特点
    1. 产生于周文疲敝之际：周文尽是外在形式的空架子
    2. 道墨两家思想范围偏重人生界
    3. 道家说道，道德根性为“无”、“反”
    4. 君人南面之术
  2. 理论体系
    1. 社会运行法则论：道
      1. 宇宙生成论
      2. 异名同物
      3. 道法自然：道是世界万物的本源，是宇宙运行的总规律
    2. 处世方法
      1. 为无为，事无事
      2. 守柔处弱
      3. 取而不争
      4. 知足
    3. 社会批判思想
      1. 反对统治阶级的剥削压迫
      2. 天道均平论：天之道损有余而补不足；圣人不积
      3. 反对礼治、法治
    4. 小国寡民论
      1. 与其说是一种社会，毋宁说是一种精神境界

庄子（古今中外最彻底之个人主义）

  1. 齐物论 庄子的遁世观
    1. 齐是非、齐万物的相对主义：
      1. 对认识客体的怀疑上，否认客观事物的本质的区别
      2. 对认识主体的怀疑上，（夏虫不可语冰）
      3. 对主客体之间的怀疑，庄生晓梦迷蝴蝶；物化
    2. 评价：
      1. 在相对和绝对的关系上,他虽然夸大了认识的相对性，丰富了认识世界的过程
      2. 有助于我们正确认识当代社会的各种思想形态,尤其是主流思想,从而不至于被一种思想模式所束缚。
      3. 庄子的相对主义在社会思想的影响的直接后果是“齐物我”；有所待
      4. 消极性：让人沉迷于神秘境界，以获得个体身心的彻底解放（qwy），塑造民族性格和劣根性——阿q胜利法
  2. 社会批判思想：
    1. ”自然人生论“：排斥社会性，人应该恪守自然本性
      1. 对治人治世的批判
      2. 对权力的批判
      3. 对名利的批判
      4. 对忠孝仁义的批判
      5. 对技术和智慧的批判
      6. 对生死观的批判
    2. 人应该摆脱名利欲望的诱惑，回归自然

墨子

  1. 思想体系：
    1. 兼爱论
      1. 强调兼，反对别；差第之爱
      2. 天下的混乱因为不相爱，不相爱因为自爱和自利
      3. 兼相爱。视人如己、爱无差等
      4. 兼相爱和交相利分不开
    2. 非命：
      1. 人与禽兽的差别，在于人是赖其力而生的（强调人的主观能动性）
    3. 天志，明鬼
      1. 天志：天志是最高裁判者
      2. 明鬼：天下混乱在于人们不信鬼神
    4. 赈灾备荒论
      1. 原因：
        1. 自然变化
        2. 横征暴敛
      2. 鉴定程度和社会影响：判断五种谷的丰歉情况
      3. 赈灾备荒方法：
        1. 积粟
        2. 节用
        3. 足财
  2. 思想评价：
    1. 发现差第之爱和平等之爱的关系
    2. 对等互报原则
    3. 对利的肯定（儒家主张利的正面作用）

韩非子

  1. 产生背景
    1. 封建到郡县
  2. 思想体系
    1. 一君万民体制
      1. 一人专制导致社会混乱的偶然性
      2. 法家不等于法制
      3. 韩非子思想的集大成性：“势法术”相统一的系统社会控制论
    2. 自利自为人性论
      1. 性恶论
      2. 性无善恶的自为人性论
    3. 反社会福利论
      1. 贫富分化合理论
      2. 反足民论
      3. 反济贫论
    4. 法术势合一的社会控制论
      1. 法：统治方法
      2. 术：权术，君主驭臣之术
      3. 势：把君主权势看作是借以行法的力量
    5. 公私观
      1. 废私立公
  3. 历史评价
    1. 强化专制，让中国古代社会长期停滞落后
    2. 儒法国家

    1. 天人合一的社会结构论
      1. 自然之天：自然界的天
      2. 人格之天：人是天的副本
      3. 神学之天：天是最高的存在
    2. 天人宇宙图式
      1. shi’wei
